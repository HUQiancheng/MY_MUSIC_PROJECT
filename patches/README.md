# Patches

Main Plugdata patches organized by function.

## Structure

- **synthesis/**: Synthesizer patches and sound generators
- **effects/**: Audio processing and effects chains
- **sequencing/**: Pattern generators, sequencers, and rhythm engines
- **composition/**: Complete tracks and compositional tools
- **experiments/**: Exploratory patches and sound design sketches

## Naming Convention

Use descriptive names with category prefixes:
- `syn-fm-bell.pd` - FM synthesis bell sound
- `fx-granular-delay.pd` - Granular delay effect
- `seq-polyrhythm.pd` - Polyrhythmic sequencer
- `comp-track01.pd` - Complete composition

## Usage

Each patch should:
1. Include abstractions from `/abstractions` folder
2. Document inputs/outputs with comments
3. Save presets for interesting parameter combinations
