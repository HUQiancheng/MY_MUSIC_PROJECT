# Samples

Audio samples and source material for patches.

## Structure

- **drums/**: Kick, snare, hi-hat, percussion samples
- **ambience/**: Pads, drones, atmospheric sounds
- **field-recordings/**: Found sounds and environmental recordings
- **processed/**: Pre-processed or mangled audio files

## Formats

- Prefer WAV or AIFF formats for quality
- Use consistent sample rates (44.1kHz or 48kHz)
- Keep original files separate from processed versions

## Organization

Name files descriptively:
- `kick-808-punchy.wav`
- `ambient-warehouse-reverb.wav`
- `glitch-vinyl-crackle.wav`
